Comprehensive CDR Analyzer - CCAT

Call Data Record Analyser for forensic investigation

June, 2016 - July, 2016
  • View project on Github
  • View report
  • Team Size: 3

Abstract

A call detail record (CDR) is a data record produced by a telephone exchange or other telecommunications equipment that documents the details of a telephone call or other communications transaction (e.g. text message) that passes through that facility or device. The record contains various attributes of the call, such as time, duration, completion status, source number, and destination number. There are two types of CDRs. One is normal CDR and other is the GPRS CDR.

Existing Technology

There are many CDR Analyzers available in the market but none of them have all the features. They only analyze the current CDR and use the older databse to give the result in a tabulated way. The Excel sheets produced are difficult to understand. The aspect of Open Source Intelligence is totally neglected in those.

We wanted to build some product that could be user friendly and processes the data fast and gives accurate results from all available open source information. We thought of adding the social media aspect as almost everyone is using them nowadays and finding a person is very easy on sites like facebook and then analyzing their accounts gives a lot of information.

Solution

A Comprehensive Call Data Record Analyzer which not only just analyzes CDRs but also use Open Source Intelligence like facebook and also plots the track of the location on Google earth. Interactive tool on excel which is very easy to use and doesn't require any coding experience to run the tool.

Features

  • Excel Sheet Converter

    CDR files from different service providers vary in terms of the data headings and formatting. The tool converts the given CDR into a standard format that it can work on and this conversion is realized with the help of a single Microsoft Excel Document with the help of Visual Basic Macros and Excel VLookUp feature.

  • Facebook Crawler

    Lists the Posts based on keyword selection ,date and time of creation. Gives all the data available publicly on facebook regarding a person's activities.

  • Analysis

    The tool provides an analyzed sheet which is easy to read and understand which contains information like frequently called numbers, unique numbers, frequently sent sms, etc. This helps in knowing information about 2nd and 3rd party involved.

  • KML generator

    Tool stores all the locations of the cell phone derived using Cell Ids, LACs, MCCs, MNCs and converts all latitudes and longitudes into a kml file.

  • Tracking using Google Earth

    Google earth plots the coordinates and get a path on it so we can trace the locations of the cell phone for the duration we are having the CDR. It works on a very kml file generated before and shows the output in a easily understandable format.

Future Enhancements

  • Output in Graphic User Interface(GUI) instead of excel.
  • Analytics along with Graph Generation.
  • Adding more Social Media crawlers like instagram, twitter, linkedin.
  • Tower Dump Analysis to find exact location of the cell phone.
